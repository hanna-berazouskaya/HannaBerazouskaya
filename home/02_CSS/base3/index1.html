Блок навигации

  • Ширина 15em
  • Должен присутсвовать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере.

Статья 1

Размер блока:
Занимает всю допустимую ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседней равным .2em, при помощи свойства margin-bottom

Статья 2

Размер блока:
Занимает всю допустимую ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседней равным .2em, при помощи свойства margin-bottom. Картинка вставленная при помощи тега, плавание организованно по правому краю внутри данной статьи для этой картинки.
Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещенным внутрь контентом.
